About Digital Election System

This system is developed by Arbind Mehta and Achyut Burlakoti , the students of Purwanchal Campus (Batch:BCT-075) Dharan, Sunsari, Nepal.

The main aim of this project is to simplify the election process of the country and to reduce the paper work during the same process. Spoilt vote is major problem of the country. This project can reduce or eliminate such issue because this project uses clickable option rather than traditional stamps. Each citizen having unique id number is eligible for creating an account. One can create account either online or by visiting related office. After creating an account one can print their confirmation page for future use. Once the date of election is announced by Election Commission Nepal , then you can vote your candidate from the specified voting booth. Make sure that your profile is approved before the election held. You can visit related office if your profile is not approved within few days or weeks. You will not be eligible to vote if your profile is not approved.

Once the election is conducted, the result can be published within a day because this project no longer uses the traditional way of counting vote. This project apply special computer program to count vote with single click. The only time consuming process is to collect databases from all the provinces.

During this process if you encounter any error regarding your profile or have any confusion then feel free to contact your VDC/Municipality office.

Note:

This project is only for experimental purpose. For now you can use any dummy data to test this project.
